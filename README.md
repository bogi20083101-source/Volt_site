## 🛠 Технологии

- **HTML5** — семантическая разметка

## 🚀 Запуск локально

Так как используются ES-модули, открывать `index.html` напрямую через `file://` **нельзя** — нужен локальный сервер.

```bash
# Вариант 1: Python
python -m http.server 8000

# Вариант 2: Node.js
npx serve

# Вариант 3: VS Code
# Установить расширение "Live Server" и нажать "Go Live"
```

Затем открыть в браузере: **http://localhost:8000**

## ☁️ Деплой на GitHub Pages

1. Создать репозиторий на GitHub (например, `volt-shop`)
2. Залить содержимое папки `volt/` в репозиторий:
   ```bash
   git init
   git add .
   git commit -m "ВОЛЬТ — итоговый проект"
   git branch -M main
   git remote add origin https://github.com/USERNAME/volt-shop.git
   git push -u origin main
   ```
3. В настройках репозитория: **Settings → Pages → Source: Deploy from branch → main /(root)**
4. Через 1–2 минуты сайт будет доступен по адресу:
   `https://USERNAME.github.io/volt-shop/`

> ⚠️ Так как используется hash-роутинг (`#/catalog`), basePath указывать не нужно — все ссылки относительные.

## 📝 Заметки по реализации

- **Авторизация** без бэкенда: пользователи хранятся в `localStorage`, токен — base64-кодированный payload (учебная имитация, не криптография).
- **Каталог** подгружается из `data/products.json` через `fetch`. Изображения — стабильно-сидированные placeholder'ы `picsum.photos`.
- **Корзина/избранное/сравнение** хранятся в едином объекте состояния (`store.js`) и автоматически сохраняются в localStorage.
- **Отзывы** имеют seed-данные (4 предзаполненных) + CRUD текущего пользователя.
- **Игра** — SVG-колесо с CSS-вращением, 3 попытки на сессию, выигрыши сохраняются.
